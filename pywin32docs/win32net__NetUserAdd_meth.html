win32net.NetUserAdd

NetUserAdd(server, level, data)

Creates a new user.

Parameters

server : string/PyUnicode

The name of the server, or None.

level : int

The information level contained in the data

data : mapping

A dictionary holding the user data in the format of PyUSER_INFO_*.
