win32file.GetFileAttributesW

int = GetFileAttributesW(fileName)

Determines a files attributes (NT/2000 Unicode specific version).

Parameters

fileName : PyUnicode

Name of the file to retrieve attributes for.

Comments

Note that win32api::GetFileAttributes will automatically call GetFileAttributesW when passed a unicode filename param. See win32file::GetFileAttributes and win32api::GetFileAttributes for more.
